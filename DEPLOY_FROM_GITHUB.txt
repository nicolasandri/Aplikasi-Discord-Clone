🔄 ROLLBACK (If Something Goes Wrong)

On VPS, run:

root@YOUR_VPS_IP:~# ls /backup/

Find the backup you want to restore, then:

root@YOUR_VPS_IP:~# rm -rf /app/frontend
root@YOUR_VPS_IP:~# cp -r /backup/frontend_backup_YYYYMMDD_HHMMSS /app/frontend
root@YOUR_VPS_IP:~# systemctl reload nginx

🐛 TROUBLESHOOTING

Problem: "Permission denied (publickey)"
Solution: chmod 600 ~/.ssh/workgrid_vps

Problem: "Command not found: curl"
Alternative: Use Git to clone instead
$ cd /tmp && git clone https://github.com/nicolasandri/Aplikasi-Discord-Clone.git
$ cd Aplikasi-Discord-Clone/app && npm install && npm run build

Problem: "npm: command not found"
Info: Node/npm might not be in PATH
Try: /usr/local/bin/npm install

Problem: "Build failed"
Try: rm -rf /app/workgrid/app/node_modules && npm install

Problem: "Connection refused"
Check: ping 152.42.229.212
Verify: SSH key with ls ~/.ssh/workgrid_vps
