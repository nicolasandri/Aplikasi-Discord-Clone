🔧 TROUBLESHOOTING

❌ "Permission denied (publickey)"
   → Check SSH key: chmod 600 ~/.ssh/workgrid_vps
   → Verify key is added to VPS: ssh-copy-id -i ~/.ssh/workgrid_vps root@152.42.229.212

❌ "Connection refused"
   → Verify VPS is running: ping 152.42.229.212
   → Check SSH is available: ssh -i ~/.ssh/workgrid_vps -v root@152.42.229.212

❌ "Not enough space"
   → Check disk: ssh -i ~/.ssh/workgrid_vps root@152.42.229.212 'df -h'
   → Clean up: rm -rf /app/frontend/*

❌ "404 on page load"
   → Verify nginx reload: systemctl reload nginx
   → Check nginx config: nginx -t
   → Verify files exist: ls -la /app/frontend/

❌ "CSS/JS not loading"
   → Clear browser cache: Ctrl+Shift+Delete
   → Check assets folder: ls -la /app/frontend/assets/
   → Check nginx error log: tail -50 /var/log/nginx/error.log
